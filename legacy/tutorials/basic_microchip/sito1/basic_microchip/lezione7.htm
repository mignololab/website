LEZIONE 7

Facciamo un sorgente (o programma) test, aprite l'IDE e scrivete quanto segue:

'Questo serve ad indicare il tipo di PIC utilizzato
'In questo caso il 16f84 e la frequenza in questo caso 4 Mhz
#chip 16f84, 4


'Fine del sorgente
END


Ora salvatelo con nome dal menù File o dal pulsante salva e compilatelo con l'icona .
Si aprirà la finestra che vi avverte della compilazione, la potete vedere in basso.





Alla fine della compilazione la finestra si chiuderà ed appariranno i file contrassegnati in rosso, nella cartella dove è presente il file COMPILE.BAT:



E' anche possibile compilare il sorgente scrivendolo con il blocco note e trascinando sullicona COMPILE.BAT il file appena creato.

Il file che deve essere scaricato all'interno del PIC è COMPILED.HEX, mentre la conversione in assembler è contenuta nel file COMPILED.ASM.
Gli altri file per il momento non ci interessano.
